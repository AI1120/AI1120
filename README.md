## 🧠 What I Do

I sit at the intersection of **Software Engineering** and **Artificial Intelligence**. My focus is on taking models from prototype to production — reliably, efficiently, and at scale.

- 🚀 **LLM Engineering**: RAG pipelines, fine-tuning (LoRA/QLoRA), agent orchestration, and prompt optimization.
- ⚡ **MLOps**: CI/CD for ML, model monitoring, automated retraining, and containerized deployment.
- 🏗️ **System Design**: Scalable APIs (FastAPI/gRPC), vector databases, event-driven architectures, and cloud infrastructure.
- 🎨 **Generative AI**: Applications powered by Transformers, diffusion models, and multimodal reasoning.
