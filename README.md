## 🧠 What I Do
I sit at the intersection of **Software Engineering** and **Artificial Intelligence**. My focus is on taking models from Jupyter notebooks to high-availability production environments.

- 🚀 **LLM Engineering:** RAG pipelines, Fine-tuning (LoRA/QLoRA), and Agent Orchestration.
- ⚡ **MLOps:** CI/CD for ML, Model Monitoring, and Automated Retraining pipelines.
- 🏗️ **System Design:** Scalable APIs (FastAPI/gRPC), Vector Databases, and Cloud Infrastructure (AWS/GCP).
- 🎨 **Generative AI:** Building applications powered by Stable Diffusion, Transformers, and Multimodal models.
